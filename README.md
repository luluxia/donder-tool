# Donder 查分器

这是一个非官方的太鼓达人查分工具，旨在帮助玩家更方便地查询和管理自己的游戏成绩。

## 功能介绍

- **曲目搜索**：快速查找游戏收录的曲目信息
- **成绩查询**：查看个人的游玩记录和详细数据
- **数据统计**：直观展示良/可/不可等判定数据
